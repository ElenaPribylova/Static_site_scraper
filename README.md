# Static Site Scraper с Аналитикой

Python web scraper для glavsnab.net с встроенным статистическим анализом цен.

## Описание

Парсит категорию товаров с сайта glavsnab.net, сохраняет данные в CSV и проводит математический анализ:
- Извлекает: артикул, название, ссылку, цену
- Анализирует: статистику цен, аномалии, частоту слов
- Визуализирует: распределение цен

## Математическая часть

### 1. Описательная статистика
Расчет основных метрик для анализа ценового диапазона:
- **Минимум/Максимум** - границы цен
- **Среднее арифметическое** - μ = Σx/n
- **Медиана** - центральное значение выборки
- **Стандартное отклонение** - σ = √(Σ(x-μ)²/n) - мера разброса
- **Квартили Q1, Q3** - 25% и 75% перцентили

### 2. Детекция аномалий (метод IQR)
Поиск подозрительных цен через межквартильный размах:
```
IQR = Q3 - Q1
Нижняя граница = Q1 - 1.5×IQR
Верхняя граница = Q3 + 1.5×IQR
```
Товары за пределами границ считаются аномалиями.

### 3. Частотный анализ
Подсчет частоты появления слов в названиях методом токенизации:
- Разбиение названий на слова
- Фильтрация стоп-слов
- Ранжирование по частоте

### 4. Визуализация
Гистограмма распределения цен с отображением среднего и медианы.

## Установка
```bash
pip install requests beautifulsoup4 lxml matplotlib
```

## Использование
```python
    parser(url="https://glavsnab.net/stroymateriali/gipsokarton.html?limit=100", 
       max_item=1132)
```

## Результаты

- `glavsnab.csv` - таблица товаров
- `analytics_report.txt` - текстовый отчет
- `price_analysis.png` - график распределения

## Пример вывода
```
Товаров: 1132 | Мин: 245₽ | Макс: 8950₽ | Средняя: 1523₽ | Медиана: 1320₽
Аномалии: низкие цены (12), высокие (8)
Топ слов: гипсокартон(456), кнауф(234), влагостойкий(189), лист(167), мм(145)
График: price_analysis.png
Отчет: analytics_report.txt
```
